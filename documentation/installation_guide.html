Beginner Python setup

Install Python, VS Code, and Git

Local setup is optional, but strongly encouraged. It lets you practice during class, try examples from the class GitHub repo, and build small side projects on your own computer.

!
Assignments are still completed in the course's web-based environments. This setup is for practice, class demonstrations, GitHub repo work, and extra projects. If installation gets difficult, keep going with the course tools and ask for help.

Setup Checklist

  1. Install Python 3

    Go to python.org/downloads and install the current Python 3 release. On Windows, check Add Python to PATH on the first installer screen.

  2. Install Visual Studio Code

    Go to code.visualstudio.com/download, download VS Code for your operating system, and run the installer.

  3. Add the Python extension

    Open VS Code, select Extensions, search for Python, and install the Microsoft extension. Direct link: Python extension.

  4. Add the Pylance extension

    In Extensions, search for Pylance and install the Microsoft extension. Direct link: Pylance extension.

  5. Install Git

    Go to git-scm.com/downloads and install Git. Restart VS Code after installation so it can detect Git.

  6. Configure Git identity

    Open Terminal, Command Prompt, or PowerShell and set the name and email address Git will attach to your commits.

Check Python

  • Open Terminal, Command Prompt, or PowerShell.
  • Try one of these commands:
python --version py --version python3 --version

Configure Git

  • Use your real name and the email address you use for GitHub.
git --version git config --global user.name "Your Name" git config --global user.email "your-email@example.com" git config --global user.name git config --global user.email

Run a Tiny Program

  • In VS Code, create hello.py.
  • Add this line, save the file, and run it:
print("Hello, Python!")

Check VS Code Git

  • Open a folder in VS Code and select Source Control.
  • Create or edit a file. VS Code should show the change.
Windows reminder If python --version does not work, try py --version. If neither works, Python may need to be added to PATH.
Mac or Linux reminder If python --version does not work, try python3 --version. Some systems keep Python 3 under that command.
Git reminder If VS Code says Git is missing, restart VS Code after installing Git. If needed, restart the computer.

When you finish: you are ready to follow class examples locally, explore the GitHub repo, track work with Git, and build small practice programs outside the required course environment.
